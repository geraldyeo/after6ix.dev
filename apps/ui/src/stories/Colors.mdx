# After6ix Design System

The After6ix design system is inspired by twilight hours when developers do their best work. Our tokens provide a consistent foundation for building beautiful, accessible interfaces.

## Usage

```tsx
// Import tokens
import { colors, spacing, typography } from '@after6ix/ui/tokens';

// Use in styles
const buttonStyle = {
  backgroundColor: colors.primary[600],
  padding: spacing[4],
  fontSize: typography.sizes.base,
};
```
