# Animations

The After6ix animation system provides smooth, performant transitions.

## Duration Scale

<div className="space-y-4 my-8">
  {Object.entries(animations.durations).map(([key, value]) => (
    <div key={key} className="flex items-center gap-4">
      <span className="text-sm font-mono text-neutral-500 w-20">{key}:</span>
      <span className="text-sm text-neutral-600">{value}</span>
      <div className="flex-1">
        <div className="relative h-2 bg-neutral-100 rounded overflow-hidden">
          <div 
            className="absolute left-0 top-0 bottom-0 bg-primary-600 rounded"
            style={{
              animation: `slide-${key} ${value} ease-in-out infinite`,
              width: '20%',
            }}
          />
          <style jsx>{`
            @keyframes slide-${key} {
              0%, 100% { transform: translateX(0); }
              50% { transform: translateX(400%); }
            }
          `}</style>
        </div>
      </div>
    </div>
  ))}
</div>

## Easing Functions

<div className="space-y-3 my-8">

      <span className="text-sm font-mono text-neutral-500 w-20">{key}:</span>

## Usage

```tsx
import { animations } from '@after6ix/ui/tokens';

// CSS-in-JS
const styles = {
  transition: `all ${animations.durations.base} ${animations.easings.out}`,
};

// Tailwind classes
<div className="transition-all duration-base ease-out">
  {/* Animation utilities map to our tokens */}
</div>
```
