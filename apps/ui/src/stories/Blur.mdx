# Blur

The After6ix blur system provides consistent backdrop blur values for creating depth and focus effects.

## Blur Scale

<div className="grid grid-cols-2 gap-4 my-8">
  {Object.entries(blur).map(([key, value]) => (
    <div key={key} className="relative h-32 rounded-lg overflow-hidden">
      <div className="absolute inset-0 bg-gradient-to-br from-primary-600 to-accent-500 p-4">
        <p className="text-white text-sm">Background content with colorful gradient</p>
      </div>
      <div 
        className="absolute inset-0 bg-white/30"
        style={{ backdropFilter: `blur(${value})` }}
      />
      <div className="absolute inset-0 flex items-center justify-center">
        <div className="bg-white rounded-lg p-4 shadow-lg">
          <div className="text-sm font-mono text-neutral-600">{key}</div>
          <div className="text-xs text-neutral-500">{value}</div>
        </div>
      </div>
    </div>
  ))}
</div>

## Usage

```tsx
import { blur } from '@after6ix/ui/tokens';

// In styles
const styles = {
  backdropFilter: \`blur(\${blur.md})\`,
};

// With Tailwind
<div className="backdrop-blur-md">
  {/* Blur utilities */}
</div>
```
