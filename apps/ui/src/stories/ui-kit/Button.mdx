## Usage Examples

### Basic Usage

```tsx
import { Button } from '@after6ix/ui/components/button';

function MyComponent() {
  return (
    <Button onClick={() => console.log('clicked')}>
      Click me
    </Button>
  );
}
```

### With Icons

```tsx
<Button>
  <DownloadIcon />
  Download Report
</Button>
```

### As Link

The button component can render as any element using the `asChild` prop from Radix UI:

<Canvas of={ButtonStories.AsChild} />

```tsx
<Button asChild>
  <a href="/dashboard">Go to Dashboard</a>
</Button>
```

## Accessibility

- All buttons have proper focus states with visible focus rings
- Disabled buttons have `aria-disabled` and prevent interaction
- Icon-only buttons should include proper aria-labels
- Keyboard navigation works with Tab and Enter/Space keys

## Design Tokens Used

- **Colors**: Primary, Secondary, Accent, Error (destructive), Neutral
- **Typography**: Font size and weight tokens
- **Spacing**: Padding uses spacing tokens
- **Shadows**: Shadow tokens for depth
- **Animations**: Transition duration tokens
- **Border Radius**: Radius tokens for corners
