# Breakpoints

The After6ix breakpoint system defines responsive design breakpoints for different screen sizes.

## Breakpoint Values

<div className="space-y-3 my-8">
  {Object.entries(breakpoints).map(([key, value]) => (
    <div key={key} className="flex items-center gap-4 p-4 bg-neutral-50 rounded-lg">
      <span className="text-sm font-mono text-neutral-600 w-16">{key}:</span>
      <span className="text-lg font-semibold text-primary-600">{value}</span>
      <span className="text-sm text-neutral-500 ml-auto">
        {key === 'sm' && 'Small devices (640px and up)'}
        {key === 'md' && 'Medium devices (768px and up)'}
        {key === 'lg' && 'Large devices (1024px and up)'}
        {key === 'xl' && 'Extra large devices (1280px and up)'}
        {key === '2xl' && '2XL devices (1536px and up)'}
      </span>
    </div>
  ))}
</div>

## Usage

```tsx
import { breakpoints } from '@after6ix/ui/tokens';

// Media queries
const styles = {
  [`@media (min-width: ${breakpoints.md})`]: {
    // Styles for medium screens and up
  }
};

// With Tailwind
<div className="w-full md:w-1/2 lg:w-1/3">
  {/* Responsive width utilities */}
</div>
```
